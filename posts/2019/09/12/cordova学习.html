# 安装

npm i cordova

# 创建项目

cordova create hello com.example.hello HelloWorld,create后边第一个参数是文件夹名，第二个是项目包名，第三个是打包后的app名

# 为cordova添加平台

cordova platform add [platform] 可选值为：android ios windows browser electron

# 系统偏好设置

  1. 触控板像windows一样，轻按点击，那么设置触控板>光标与点按>轻点来点按
  2. 为不同的应用设置touchbar，键盘>快捷键>功能键，点击添加按钮来添加应用。
  3. 双屏时固定dock位置：系统偏好设置>调度中心>显示器具有单独的空间勾选去掉，然后注销系统

# 程序坞

  1. 修改程序坞的位置：按住shift键点击拖动
  2. 移除程序坞中的应用：鼠标按住图标移到程序坞外，或者拖拽到垃圾桶
  3. 防止误删除：右键应用图标>选项>在程序坞中保留
  4. 连按应用程序标题栏最小化：程序坞偏好设置>连按窗口标题栏以>选择最小化

# 访达

  1. 改变工具栏显示位置：command+拖动
  2. 删除工具栏图标：command+移出窗口
  3. 显示文件夹有多少文件和可用空间：显示>显示状态栏
  4. 显示文件夹路径：显示>显示路径栏
  5. 返回上层文件夹：command+⬆
  6. 桌面显示硬盘：访达>偏好设置>通用>硬盘
  7. 显示文件扩展名：访达>偏好设置>高级>显示所有文件扩展名
  8. 标记（为文件分类）：文件右键>标记，还可以在偏好设置中修改标记名称
  9. 永久显示隐藏文件 defaults write com.apple.finder AppleShowAllFiles TRUE(回车) killall Finder(回车)

# 触发角

mac系统四个角可以设置为触发角，来快速显示某些设置或者界面

  1. 触发角设置：系统偏好设置>调度中心>触发角
  2. 防止触发角误触发：在选择触发角时，可以通过按住键位来添加组合键，比如command、option、shift、control

# 其他

  1. 安装外部应用：下载之后，双击dmg文件，之后拖动到Applications中即可
  2. 安装第三方应用：系统偏好设置>安全与隐私>通用>点击锁按钮>任何来源，在新版mac系统中隐藏了该选项，可以通过终端>sudo spctl --master-disable
  3. 卸载应用程序：访达>应用程序>把应用拖到废纸篓

# 新机设置

  1. 触控板像windows一样，轻按点击，那么设置触控板>光标与点按>轻点来点按
  2. 提升打字速度和删除文字速度：键盘>按键重复和重复前延迟>拉到最右
  3. 削弱动画：辅助功能>显示>减弱动态效果
  4. 永久显示隐藏文件 defaults write com.apple.finder AppleShowAllFiles TRUE(回车) killall Finder(回车)
  5. 显示文件扩展名：访达>偏好设置>高级>显示所有文件扩展名
  6. 显示文件夹路径：显示>显示路径栏
  7. 显示文件夹有多少文件和可用空间：显示>显示状态栏
  8. 双屏时固定dock位置：系统偏好设置>调度中心>显示器具有单独的空间勾选去掉，然后注销系统

# Read-only file system

这是因为SIP保护机制的问题，解决方法：

  • 重新启动，重启时按住 Command+R直到出现苹果的图标和进度条
  • 进入左上角实用工具>终端
  • 输入csrutil disable回车
  • 重启后执行sudo mount -uw /
  • 现在可以在根目录下新建文件夹了，但是还是要使用sudo命令

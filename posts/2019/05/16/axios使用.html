# axios get的params怎么传入数组

默认params传入数组的话，传值的时候会变成下面这样

axios.get(url,{
  params:{
    tag:[1,2,3,4]
  }
})
// ...tag[]=1&tag[]=2&tag[]=3&tag[]=4

如果真的是要tag=1&tag=2，那就...

axios.get(url,{
  params:{
    tag:[1,2,3,4]
  },
  paramsSerializer: function(params) {
    const yourNewParams = params.tag.map(_=>`tag=${_}`).join('&')
    return yourNewParams
  },
}) 
// ...tag=1&tag=2&tag=3&tag=4

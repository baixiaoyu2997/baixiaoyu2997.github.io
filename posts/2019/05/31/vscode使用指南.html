# snippets代码段

  1. 如果想要智能提示中代码段排在最上面，需要设置"editor.snippetSuggestions": "top"
  2. 输入特殊字符时需要使用\\转义
  3. 换行\n
  4. tab键制表符：\t
  5. 如果想光标选中字符则${1:test}

# 配置假名路径的智能提示和跳转

把jsconfig.json文件放到根目录下，记得添加到.gitignore上

// jsconfig.json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    },
    "target": "ES6",
    "module": "commonjs",
    "allowSyntheticDefaultImports": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}

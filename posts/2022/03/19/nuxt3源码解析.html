# nuxi

主文件cli.ts, index.ts文件为内部使用

# cli

功能点：

  1. 检查node版本
  2. 全局handle错误
  3. init:初始化项目，使用tiged从github下载模板
  4. dev：启动开发模式

# edge

通过github workflow执行脚本release-edge发布

# 流程图

# 创建项目

nuxi init nuxt3-app

# dev

  1. nuxi启动命令行

  2. 创建服务

  3. 监听项目根目录变化

  4. 加载nuxt3、nuxt-edge、nuxt之一：

    1. 加载config
    2. 创建nuxt对象，并返回

  5. 创建nuxt对象，初始化hooks，返回nuxt对象

  6. 等待ready(),也就是nuxt初始化：

    1. hooks添加config中的hooks
    2. 初始化nitro
